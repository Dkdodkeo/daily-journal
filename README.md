я создал первый свой проэкт с довольно большой частью которого является логика js кода 
здесь я создал приложение по теме ежедневной записной книжки,проще говоря ежедневник.
также для удобства образного клиента я добавил смену темной и ночной темы

множество хорошей и приятной глазу hover анимции 

This is my very first project, and a big part of it is the JavaScript logic. I created an application in the form of a daily journal — in other words, a personal planner.
✨ Features I added for convenience and user experience:
- Light/Dark theme switcher for comfortable use at any time of day.
- Persistent storage: the existing list is saved even after refreshing the page (via localStorage).
- Smooth hover animations that make the interface pleasant and visually engaging.
The goal was not only to practice JavaScript, but also to build something useful and visually appealing.
